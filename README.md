# Python Mini Projects – Internship Submission

This repository contains three beginner-friendly Python projects built as part of the internship program.

## Projects

### 1. To-Do List (`todolist.py`)
A simple command-line to-do list application.

**Features:**
- Takes 3 tasks as input from the user
- Displays the final list with numbering

**How to run:**
```bash
python todolist.py
```

---

### 2. Expense Tracker (`expensetracker.py`)
A basic expense tracker that calculates total spending.

**Features:**
- Accepts 5 expense entries from the user
- Calculates and displays the total expense

**How to run:**
```bash
python expensetracker.py
```

---

### 3. Password Generator (`passwordgenerator.py`)
A secure random password generator.

**Features:**
- User specifies desired password length
- Ensures the password contains at least one uppercase letter, one lowercase letter, and one digit
- Randomly shuffles characters for added security

**How to run:**
```bash
python passwordgenerator.py
```

---

## Requirements
- Python 3.x
- No external libraries required (uses built-in `random` and `string` modules)
